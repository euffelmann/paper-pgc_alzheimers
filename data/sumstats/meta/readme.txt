this directory contains the summary statistics for the set of meta-analysis in
PGC-ALZ3

- fema: female-only
- male: male-only
- main: main meta-analysis
- newc: meta-analysis of all new cohorts compared to PGC-ALZ2
- xchr: sumstats for chromosome x only
- loo.R: script to generate leave-one-cohort-out summary statistics
