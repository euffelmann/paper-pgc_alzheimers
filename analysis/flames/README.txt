This folder contains the analysis commands used to run the FLAMES analysis in this manuscript.
Please not that this is pseudocode written for python, on a linux system. Other setups may need a rewrite of the code.

Paths have been rewritten and need to be adjusted for own system.
All necessary data outside of summary statistics can be found on https://zenodo.org/records/12635505.
For more information on FLAMES please visit the GitHub.

This code has been written in 3 sections. Each section requires a different conda environment and installation of tools. 
Section 1 requires a conda environment for, and installation of PolyFun: https://github.com/omerwe/polyfun

Section 2 requires a conda environment for, and installation of PoPS: 
Additionally you need the PoPS features downloaded from https://zenodo.org/records/12635505. Note that you only need the FUMA compatible full feature set to repeat this analysis.

Section 3 requires a conda environment for, and installation of FLAMES: https://github.com/Marijn-Schipper/FLAMES

For issues or questions, please contact the corresponding authors or contact m.j.schipper@vu.nl directly.
